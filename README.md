# Data-Analytics-Day1   This dataset contains information about 10 students, including Student ID, Student Name, Course, Fees, and Discount Percentage.

The following calculations and analysis have been performed using Excel formulas:

SUM (Total Fees): Calculated the total fees collected from all students by adding their fee amounts.
AVERAGE (Average Fees): Determined the average fee amount of all students.
MIN (Minimum Fees): Identified the lowest fee value in the dataset.
MAX (Maximum Fees): Identified the highest fee value in the dataset.

Discount Calculation: Calculated the discount amount for each student based on their fee and discount percentage using the formula:

Discount Amount = Fees × Discount Percentage ÷ 100

This analysis helps in understanding the overall fee structure, identifying fee ranges, and determining the discount provided to each student. It also demonstrates the use of basic Excel functions for data analysis and calculation.

Short Description

This student fee dataset was analyzed using Excel functions such as SUM, AVERAGE, MIN, and MAX to calculate total fees, average fees, minimum fees, and maximum fees. Additionally, discount amounts were calculated for each student based on their respective discount percentages. The analysis provides a clear summary of fee data and demonstrates the practical use of Excel formulas for data management and reporting.
